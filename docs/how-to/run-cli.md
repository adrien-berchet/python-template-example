# Run the Command-Line Interface

The generated project exposes a small Click-based command-line interface.

To see the installed version:

```bash
python-template-example --version
```

To run the example command:

```bash
python-template-example -x 1 -y 2
```

You can also invoke the package module directly:

```bash
python3 -m python_template_example.cli -x 1 -y 2
```
